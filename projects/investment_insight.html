Investment Insight

10.07.2020

Github Repository might be made public in the future

Description

The goal of this project is to create a dashboard for the analysis of stock data. For a chosen company, the dashboard will show historical prices and key performance indicators as well as the results of Advanced Analytics being applied on the historical data. I am building the dashboard to give usefull information about a given stock. I do not plan to do any algorithmic trading. The results of my analysis of the key performance indicators and the historical stock prices will not trigger automatic Buy or Sell decisions, but instead will just be visualized in the dashboard. This way I can look at the dashboard of a company that I find interesting and see whether or not Now is a good time to buy according to the historical performance indicators and according to strategies being executed on the historical stock prices. The scope of the project is to aid with investing decisions with a time horizont of months to years. The project has three major technical components:

  • Investment Insight: Data is responsible for collecting data from different data sources, safe the data in an SQL database and vendor it to both the Dashboard directly (if it is data that I will publish raw) and to the Analytics component.
  • Investment Insight: Analytics is responsible for gathering insights from the data using statistics and Machine Learning.
  • Investment Insight: Dashboard is responsible for the actual visualization of the results.
The focus of the project will be on the german stock indices. I am especially interested in smaller companies in the tech industry. The first version of the dashboard is going to include just a small number of companies and a small number of dashboard elements. This way I can try out the whole pipeline from data collection to deployment without getting too engaged with any of the technologies used. I am going to containerize the entire code base so that I have more freedom in regards to where each of the three components is running.
